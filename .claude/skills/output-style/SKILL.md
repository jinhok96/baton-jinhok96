## 언어

모든 응답, thinking, 분석은 **한국어**. 영어 사용 금지.

## 파일 참조 형식

파일이나 코드 위치 언급 시 반드시 클릭 가능한 마크다운 링크 사용:

- 파일: `[filename.html](path/to/filename.html)`
- 특정 라인: `[filename.html:42](path/to/filename.html#L42)`
- 범위: `[filename.html:42-51](path/to/filename.html#L42-L51)`
